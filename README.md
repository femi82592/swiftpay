# SwiftPay
SwiftPay is a web-based payment and tipping platform built with HTML and deployed on Vercel. The application features a streamlined payment flow with QR code generation for tip transactions, a user onboarding join flow, and a privacy policy. Built and iterated across 15 commits with 14 production deployments.
Live demo: https://swiftpay-orcin.vercel.app
Tech stack: HTML, JavaScript, Vercel
Features:

QR code based tip URL generation
User join and onboarding flow
Production deployed and publicly accessible
